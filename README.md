# Aqsa Birthday Page

This is a static birthday page ready for Vercel deployment.

## Deploy with Vercel

### Option 1: Drag and drop
1. Open https://vercel.com/dashboard.
2. Click `New Project`.
3. Drag the `aqsa-main` folder into the browser.
4. Confirm the project settings and deploy.

### Option 2: Vercel CLI
1. Install Vercel CLI:
   ```bash
   npm install -g vercel
   ```
2. Deploy from the project folder:
   ```bash
   cd /home/sanskar/Downloads/aqsa-main
   vercel
   ```
3. Follow the prompts and choose the current folder.

## Notes
- `index.html` is the entry point.
- All static assets are inside the `images/` folder.
- The deploy should work without a build step.
